Requirements
============

+ PHP >= 5.6
+ Node.js
+ Bower
+ Symfony 2.8
+ Git
+ Composer


Installation
------------

$ git clone https://github.com/vmolero/semantic-search.git vmolero

$ cd vmolero

$ chmod +x run.sh

$ ./run.sh (run.bat if windows)

Open browser and go to:

http://localhost:8000/rate/new
